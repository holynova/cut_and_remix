# 织影 WeaveArt - 数字化照片编织艺术工坊

WeaveArt 是一款网页版的数字化照片编织（Photo Weaving）与图像切片重组的艺术创作工具。该项目旨在帮助用户在线生成极具质感、解构主义风格的编织艺术作品。

## 🎨 核心功能特点

1. **双层经纬网格编织 (Grid Weave)**
   - 支持经线（Warp - 纵向纸条）与纬线（Weft - 横向纸条）的格子交叉穿插。
   - 纬线支持使用 **第二张图片**（进行双图混织，如肖像与风景交错）或 **单色/多色卡纸填充**。

2. **多重编织交织方式**
   - **平纹编织 (Plain Weave)**：经典的 1x1 交错编织。
   - **斜纹编织 (Twill Weave)**：提供 2x2 与 3x3 的斜向阶梯式错位效果。
   - **对角线式交织**：极具几何构图美感的斜纹遮挡。

3. **竖条波形错位 (Vertical Wave Shift)**
   - 将图像切成等宽垂直条带，支持正弦波、交替上下、锯齿波与伪随机波形等多种数学算法进行垂直位移，产生类似“瓦楞折射”的动态波浪肌理。

4. **拟真 3D 物理质感**
   - **折叠阴影 (Drop Shadow)**：根据交叉层级关系，在交汇处动态渲染微投影，模拟纸条叠压的真实深度。
   - **切口反光 (Edge Highlight)**：在切片左右或上下边缘绘制轻微的高光，模拟手工裁切纸张的横截面质感。
   - **纸张肌理 (Paper Texture)**：支持叠加程序化生成的 **哑光纸**、**粗糙画布** 与 **细密亚麻布** 等纹理滤镜。

5. **流畅的交互体验**
   - 交互画布支持鼠标滚轮或触控板手势进行平滑的 **放大缩小 (Zoom)** 动作，以及左键拖拽 **平移 (Pan)**。
   - 内置官方样例，一键加载预设并配置最匹配的参数以呈现最佳视觉效果。

## 🛠️ 技术实现

- **核心语言**：HTML5 / Modern CSS (CSS 变量、磨砂玻璃拟物化设计) / Vanilla JavaScript (ES6)
- **渲染引擎**：纯客户端 HTML5 Canvas 2D API 硬件加速渲染，不依赖任何第三方重量级框架，启动速度极快。
- **程序化材质**：通过离屏 Canvas 渲染随机噪点或像素纹理，使用正片叠底 (`multiply`) 混合模式实现无图片依赖的高性能纸张纹理叠加。

## 🚀 本地快速启动

由于本项目完全由原生静态网页技术开发，您可以在本地直接使用任何轻量级静态服务器运行：

### 1. 使用 Node.js (推荐)
```bash
# 进入项目目录
cd cut_and_remix

# 使用 npx 快速启动
npx http-server -p 8080
```

### 2. 使用 Python
```bash
# Python 3
python3 -m http.server 8080
```

启动后在浏览器中打开 [http://127.0.0.1:8080](http://127.0.0.1:8080) 即可开始创作。
